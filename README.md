### [dao-bi](https://github.com/Suzzt/dao-bi)
一个开源的商业智能（BI）报告生成工具，旨在通过拖拽的方式快速生成 BI 报告。支持多种数据源和复杂的数据处理逻辑，帮助用户快速构建数据分析报告。为什么要写？？？因为想实现人人是数据分析师的愿景！
